# State-Level Greenery Management System (ERP)
## Professional Handover Documentation v1.0

Welcome to the **SLGMS ERP**, a high-fidelity enterprise platform designed for managing university-wide greenery projects.

### 1. System Overview
The platform is built on a custom **Glassmorphism Design System** using PHP 8.x, MySQL, and Bootstrap 5. It features a fully responsive administration dashboard with real-time data visualization.

### 2. Core Modules
- **Analytics Dashboard:** Real-time KPIs, Growth Trends (Chart.js), and Activity timelines.
- **Lawn Management:** Masonry-grid categorization with weighted health scoring (Best/Average/Worst).
- **Botanical Herbarium:** Detailed plant tracking with scientific classification and vitality monitoring.
- **Water & Irrigation:** IoT-ready zone control with consumption analytics.
- **Workforce Hub:** Staff performance tracking and lawn assignment monitoring.
- **GIS Mapping:** Interactive campus master plan with zone markers.
- **Advanced Reporting:** Generation of PDF audits and Excel (CSV) data exports.

### 3. Deployment Instructions
#### Local Setup (XAMPP/WAMP)
1. **Database:** Create a database named `greenery_management` and import the `database/greenery_db.sql` file.
2. **Configuration:** Ensure `includes/db.php` has the correct credentials (default: localhost, root, no password).
3. **Permissions:** Ensure the `assets/uploads/` directory is writable.
4. **URL:** Access via `http://localhost/State Level Greenery Management System/`.

### 4. Administrative Credentials
| Role | Username | Default Password | Access Level |
| :--- | :--- | :--- | :--- |

### 5. Security Protocols
- **RBAC:** Role-Based Access Control is enforced via `requireRole()` and `hasPermission()`.
- **CSRF Protection:** All POST requests require a valid security token generated by the system.
- **Data Sanitization:** All user inputs are sanitized using the `sanitize()` utility in `functions.php`.

### 6. Design System Tokens (CSS)
- **Primary Color:** `#2d6a4f` (Deep Forest Green)
- **Accent Color:** `#52b788` (Vibrant Emerald)
- **Glass Effect:** `blur(15px)` with `rgba(255,255,255,0.03)` background.

### 7. Future Scalability
- **IoT Integration:** The `irrigation_zones` table is ready for API hooks to real-time soil moisture sensors.
- **GIS Mapping:** The `map-management.php` explorer can be upgraded to Leaflet.js or Google Maps API using the existing coordinate markers.
